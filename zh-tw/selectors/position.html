# position value

  • static
  • relative
  • absolute
  • fixed
  • sticky

預設是 static 不能移動

relative 可移動

absolute 可移動 新圖層

fixed 可移動 釘死在畫面 top 按鈕

sticky 可移動 黏住 導航 nav

先往外層的元素找，否則定位點在左上角
